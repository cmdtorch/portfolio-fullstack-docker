## Deployment

### Clone repository
```bash
$ git clone https://github.com/cmdtorch/portfolio_api
```

### Run build.sh file 
You need to fill in all fields for the project build with your data
```bash
$ sh build.sh
$ ...
```

### Build and Run docker

```bash
$ docker compose up --build
```
